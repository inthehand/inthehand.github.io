The Natural Language field of the entry. Some example values are 0x656E which is "en", and 0x6672 which is "fr".
encodingId
Type: SystemUInt16
The IETF Charset identifier for this language. e.g. 3 for US-ASCII and 106 for UTF-8, see EncodingId
baseAttributeId
Type: InTheHand.Net.BluetoothServiceAttributeId
The base Attribute Id for this language in the record. e.g. 0x100 for the Primary language.
