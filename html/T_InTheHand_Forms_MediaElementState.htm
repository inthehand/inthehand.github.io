Members
  Member nameValueDescription
Closed0 The MediaElement contains no media. The MediaElement displays a transparent frame.
Opening1 The MediaElement is validating and attempting to load the specified source.
Buffering2 The MediaElement is loading the media for playback. Its Position does not advance during this state. If the MediaElement was already playing video, it continues to display the last displayed frame.
Playing3 The MediaElement is playing the current media source.
Paused4 The MediaElement does not advance its Position. If the MediaElement was playing video, it continues to display the current frame.
Stopped5 The MediaElement contains media but is not playing or paused. Its Position is 0 and does not advance. If the loaded media is video, the MediaElement displays the first frame.
